# Team Management Portal Backend Starter Kit

Node.js + Express + SQL Server backend starter kit for a Team Management and Task Tracking Portal.

## Features
- JWT Authentication
- Modular architecture
- SQL Server schema
- Winston logging
- asyncHandler utility to avoid repetitive try/catch
- Global error handling

## Setup
1. Install dependencies:
   npm install
2. Copy `.env.example` to `.env`
3. Update database credentials
4. Run SQL script in `database/schema.sql`
5. Start server:
   npm run dev
